Computer Graphics Quiz - Geometric Modeling

1. Which of the following is NOT a Platonic solid?

2. In CSG (Constructive Solid Geometry), which operations can be performed with objects?

3. What parameters are used to define points on a sphere in parametric form?

4. In a solid polyhedron, face normals must be specified in counterclockwise order.


5. For a sphere centered at the origin, every point on the sphere can serve as its own normal vector.


6. Match each geometric concept with its characteristic:

Platonic Solid
Parametric Sphere
Cylinder Approximation

Answer Key:

Multiple Choice Questions:

1. c - Cylinder (It's not a Platonic solid as it has curved surfaces)

2. c - Union, subtraction, and XOR are all possible operations in CSG

3. b - Latitude (φ) and longitude (θ) are used to define points on a sphere

True/False Questions:

4. True - Face normals must be specified in counterclockwise order for proper rendering

5. True - For a unit sphere centered at the origin, each point's coordinates can serve as its normal vector

Matching Question:

Platonic Solid → All faces are same size and shape

Parametric Sphere → Uses latitude and longitude

Cylinder Approximation → Built from vertical strips

Explanations:

1. Cylinders have curved surfaces and are not Platonic solids, which must have regular polygon faces.

2. CSG allows multiple boolean operations between geometric objects.

3. Spherical coordinates use latitude and longitude to specify points on a sphere's surface.

4. Counterclockwise ordering is important for correct face culling and lighting calculations.

5. This property makes normal calculations for spheres particularly simple.

6. Each geometric concept has distinct characteristics that define how it's constructed and represented.
