Answer Key:

Multiple Choice Questions:

1. b - Color buffer, depth buffer, stencil buffer, and accumulation buffer

2. b - To composite multiple images and create high-quality effects

3. b - Scissor test, alpha test, stencil test, depth test

4. b - To implement transparency by combining new and old color values

5. b - To sample more pixels than displayed and average them for smoother edges

9. b - To slightly adjust polygon depth to avoid z-fighting

10. b - To approximate colors by mixing darker and lighter pixels

True/False Questions:

6. True - Order matters for correct transparency

7. False - The stencil buffer can be modified during rendering

Matching Questions:

8. Depth Buffer → Stores z-depth information

8. Stencil Buffer → Enables masking operations

8. Accumulation Buffer → Composites multiple images

Explanations:

1. The frame buffer consists of multiple components each serving different purposes.

2. The accumulation buffer is used for effects requiring multiple image combination.

3. Fragment operations follow a specific order in the OpenGL pipeline.

4. Alpha blending enables transparency effects by combining colors.

5. Multisampling improves image quality by averaging multiple samples.

6. Render order is crucial for correct transparency effects.

7. The stencil buffer can be dynamically modified during rendering.

8. Each buffer type serves a specific purpose in the rendering pipeline.

9. Polygon offset helps avoid z-fighting in overlapping geometries.

10. Dithering is used to approximate colors on limited color displays.
