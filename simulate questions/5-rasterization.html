Computer Graphics Quiz - Rasterization

1. What is the main problem with using explicit form for line drawing?

2. In barycentric coordinates (α,β,γ), what is always true?

3. Which form is most suitable for color interpolation in line drawing?

4. In triangle rasterization, the winding order (clockwise or counterclockwise) affects which side is considered "inside" the triangle.


5. The half-plane test divides space into three regions when testing if a point is inside a triangle.


6. Match each rasterization approach with its key characteristic:

Explicit Form
Implicit Form
Parametric Form

Answer Key:

Multiple Choice Questions:

1. b - It doesn't work for slope > 1 and requires special handling

2. c - α + β + γ = 1

3. c - Parametric form

True/False Questions:

4. True - Clockwise winding means inside is on the right, counterclockwise means inside is on the left

5. False - The half-plane test divides space into two regions for each line of the triangle

Matching Question:

Explicit Form → Uses slope-intercept equation

Implicit Form → Uses distance from line

Parametric Form → Uses time parameter

Explanations:

1. The explicit form requires different handling for slopes > 1 and cases where run = 0, making the implementation more complex.

2. In barycentric coordinates, the sum of all coordinates must equal 1, which ensures proper interpolation.

3. Parametric form makes color interpolation straightforward as it naturally handles the progression from start to end point.

4. The winding order is crucial for determining which side of each edge is considered inside the triangle.

5. The half-plane test uses three line tests to determine if a point is inside a triangle, with each line dividing space into two regions.

6. Each form has its characteristic approach: explicit uses y=mx+b, implicit uses distance measurements, and parametric uses a time parameter t.
